# The-Castle
Short Unity made 3D horror game.
## Objective
Find and collect 4 hidden keys all around the castle while avoiding the monster.<br>After you find them exit through the main door.
## Controls
| Action      | Button    |
|-------------|:---------:|
| Move        | WASD keys |
| Look around | Mouse     |
| Interact    | Let-click |
| Pause/Menu  | Esc       |
